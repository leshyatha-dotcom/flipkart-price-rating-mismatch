# Flipkart Price-Positioning & Rating Mismatch Analysis

## Problem Statement
Customers often can't tell if a product's price reflects its actual
quality. This project identifies overpriced brands (high price, low
rating) vs hidden gems (low price, high rating), and checks whether
"discounts" are inflated through fake MRP markups.

## Dataset
Flipkart Products Dataset — Source: Kaggle (link)

## Approach
1. Cleaned pricing and rating columns
2. Calculated Value Score = rating relative to normalized price
3. Identified top "hidden gem" and "overpriced" products
4. Analyzed category/brand-level pricing patterns
5. Checked MRP vs Discount% correlation for fake-discount signals
